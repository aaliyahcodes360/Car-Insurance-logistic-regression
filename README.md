# Car-Insurance-logistic-regression
Python data analysis project using logistic regression to predict car insurance outcomes.
## Project Overview
This project explores which factors best predict car insurance outcomes using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Statsmodels

## Project Steps
- Loaded the dataset using Pandas
- Handled missing values by replacing them with the mean
- Built logistic regression models for each feature
- Evaluated models using confusion matrices
- Compared model accuracy

## Goal
To identify which feature best predicts the car insurance outcome.
